# learn-system
【2026原创】基于Vue3的在线学习刷题平台 系统实现


## 一、介绍
在线学习刷题平台系统，本项目是一款基于Vue3技术栈开发的在线学习刷题系统，旨在为用户提供便捷的在线学习和练习平台。系统采用前后端分离架构，后端使用Flask框架提供RESTful API服务，前端采用Vue3框架构建用户界面。系统分为用户和管理员两个角色：用户可以进行用户注册登录、在线刷题练习、查看练习记录、错题回顾以及题目收藏等功能；管理员则负责系统内容的管理，包括用户管理、题目上传与编辑、分类管理等。系统支持单选题（A、B、C、D选项）和填空题两种题型，用户每次练习可以自定义题目数量和分类，系统会智能排除用户已掌握的题目，实现个性化学习。此外，系统提供练习准确率统计、答题详情展示等数据分析功能，帮助用户了解自己的学习进度和薄弱环节。
